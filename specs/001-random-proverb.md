# Goal

Build a very small web app that shows one random proverb on the page.

# Requirements

- The page must display a title: Random Proverb
- The page must show one proverb chosen randomly from an in-code list
- There must be a button labeled "Show another proverb"
- Clicking the button must replace the proverb with another random proverb
- No database
- No authentication
- Keep it very simple

# Constraints

- Use a minimal stack suitable for a beginner solo project
- Keep all data local in the codebase
- Keep the project easy to deploy later

# Acceptance criteria

- App starts locally
- Visiting the page shows a proverb
- Clicking the button changes the proverb
